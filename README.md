# 🍎 No Added Sugar — March 2026

A simple, beautiful daily tracker to help you cut out added sugar for the month of March.

## Features

- 📅 Full March calendar view
- ✓ Mark days as clean or slipped
- 🔥 Streak counter
- 📊 Progress bar & stats
- 💾 Auto-saves to your browser (persists after closing)
- 🍎 Natural sugars & fruit are allowed!

## How to Use

1. Open `sugar-tracker.html` in any browser
2. Tap a day to cycle through: **clean ✓** → **slipped ✗** → **unmarked**
3. Your progress saves automatically

## Hosting

This is a single HTML file — no build step, no dependencies. You can host it anywhere:

- **GitHub Pages** — push to a repo and enable Pages in Settings
- **Netlify Drop** — drag and drop at [netlify.com/drop](https://netlify.com/drop)
- **Locally** — just open the `.html` file in your browser

## Tech

Plain HTML, CSS, and JavaScript. No frameworks, no dependencies. Data is stored in `localStorage`.

## Goal

> Cut out added sugar for all of March 2026. Natural sugars and fruit are welcome. 🍏
